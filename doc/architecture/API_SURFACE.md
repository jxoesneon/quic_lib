# API Surface Architecture

The authoritative Dart API surface is specified in [DART_API_SPEC.md](../specs/DART_API_SPEC.md).

This document previously contained a class hierarchy overview. All class definitions, signatures, and behavioral contracts have been consolidated into the specification above to eliminate duplication.

For the HTTP/3 API, see [DART_API_SPEC.md §2.6](../specs/DART_API_SPEC.md#26-http3-api).
For the WebTransport API, see [DART_API_SPEC.md §2.7](../specs/DART_API_SPEC.md#27-webtransport-api).
For the libp2p API, see [DART_API_SPEC.md §2.8](../specs/DART_API_SPEC.md#28-libp2p-api).
